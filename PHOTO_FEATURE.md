# Функція фото товарів

## Опис

Функція дозволяє користувачам завантажувати до 3 фотографій для кожного товару.

Фото відображуються:
- На картці товару (стікер з кількістю, наприклад: 📷 2)
- У модальному вікні додавання товару (фотогалерея)

**Важливо:** фото використовуються тільки для візуалізації при додаванні товарів у список. Вони **не включаються** в звіти, статистику або експортовані файли.

## Статус реалізації

Функція реалізована та інтегрована у `main` (бекенд + фронтенд + UI модерації в адмінці).

## Де в коді

- UI та підключення: `webapp/templates/index.html`
- Логіка на фронтенді: `webapp/static/js/photos.js`
- Стилі: `webapp/static/css/photos.css`
- API (FastAPI): `webapp/routers/photos.py`
- Стискання зображень: `webapp/utils/image_processing.py`

## Основні можливості

### Для користувачів

- Завантаження фото: максимум 3 фото на товар, автостискання до ~500KB, підтримка популярних image-форматів.
- Перегляд фото: гортання між фото, повноекранний режим, лічильник фото (1/3, 2/3, тощо).
- Візуальні індикатори: стікер на картці товару показує кількість фото.

### Для адміністраторів

- Модерація фото: черга `pending`, швидке схвалення/відхилення, причина відхилення.
- Керування фото: видалення недопустимих фото, перегляд базової статистики, відстеження авторів завантажень.

## Технічні деталі

### Обмеження

- Максимум 3 фото на один товар.
- Максимальний розмір завантажуваного файлу: 10MB.
- Схвалені фото відображаються всім, `pending/rejected` — ні.

### Стискання зображень

- Максимальний розмір: 1200px по більшій стороні.
- Цільовий розмір файлу: ~500KB.
- Формат: JPEG з оптимізацією.

### API ендпоінти

#### Завантаження фото

```
POST /api/photos/upload
Content-Type: multipart/form-data

Body:
- photo: File
- article: string
- user_id: integer
```

#### Отримання фото товару (тільки approved)

```
GET /api/photos/product/{article}

Response: {
  success: true,
  photos: [
    {id: 1, file_path: "...", order: 0},
    ...
  ]
}
```

#### Модерація

```
GET /api/photos/moderation/pending?user_id={admin_id}
POST /api/photos/moderation/{photo_id}
  - status: "approved" | "rejected"
  - reason: string (optional)
  - user_id: integer
```

#### Видалення фото

```
DELETE /api/photos/{photo_id}?user_id={admin_id}
```

## Використання

### Для користувачів

1. Відкрити модальне вікно додавання товару.
2. Натиснути "📷 Додати фото".
3. Обрати фото з галереї або зробити нове.
4. Фото автоматично стиснеться і надішлеться на модерацію.

### Для адміністраторів

1. Відкрити вкладку "🛠️ Адмін".
2. Прокрутити до секції "🖼️ Модерація фото".
3. Схвалити або відхилити фото (за потреби вказати причину).
