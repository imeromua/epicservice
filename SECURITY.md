# Security Policy

## Supported Versions

Проєкт використовується переважно для внутрішніх задач. Підтримуваною вважається остання версія в `main` та останній опублікований реліз.

## Reporting a Vulnerability

Якщо ви знайшли вразливість:

1. Не створюйте публічний issue з деталями експлойту.
2. Напишіть приватно на email: [imerom25@gmail.com](mailto:imerom25@gmail.com)
3. Або в Telegram: @my_life_ukr

У листі бажано вказати:
- Кроки для відтворення
- Очікувану/фактичну поведінку
- Потенційний вплив
- Версію/коміт
